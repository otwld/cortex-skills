# Project Memory

Project memory is durable context for agents. Use CONTEXT.md for a single domain
glossary, CONTEXT-MAP.md for multiple contexts, docs/adr for decisions, .out-of-scope
for rejected enhancements, docs/agents/issue-tracker.md for tracker mode, and
docs/agents/triage-labels.md for label mapping. Create files lazily and never overwrite
existing memory without approval.
